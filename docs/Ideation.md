# 취업 포트폴리오 해커톤 아이디어

## Day1

### 동기부여 세션 - AI를 내 편으로 만들자!
- AI의 등장이 혁신을 가속화 하고 있음
  - 개발자와 데이터 분석가등의 직무가 AI로 대체될 가능성이 매우 높음
- 과거의 성공 사례가 큰 도움이 되지 않음
  - 학력, 자격증은 의미가 없음
  - 교수님들 포함해서 부모세대의 경험이 큰 도움이 되지 않을 수 있음
- AI를 두려워 할 것이 아니라 내 편으로 만들자!
  - 누구보다 더 AI를 잘 활용하는 사람이 되자
- Apply Driven Development
    - 일하고 싶은 채용 공고를 찾는다.
    - 좋은 채용공고는 어떤 일을 하는지, 어떤사람을 선호하는지가 적혀 있음.
        - 좋은 채용공고의 예시: https://samyangroundsquare.ninehire.site/job_posting/N0n86hFP
    - 퍼플렉시티 또는 구글 AI 검색을 사용
    - 꼭 국내 회사만 고집할 필요는 없다! 세계 최고 클래스의 일자리를 목표로 하자!
    - 채용공고의 내용을 반영한 프로젝트를 기획하고 구현한다.
    - 기술 스택, 업무 내용등을 반영
    - 구현과 학습을 동시에 진행 
      - 모르는것은 AI에게 물어보고, 이해한것이 맞는지를 AI에게 설명하여 확인 받는다.
    - 이 과정을 반복하면 포트폴리오를 빠르게 쌓아 나갈 수 있음.
    - 학생이라도 시작해보자!
      - 원하는 취업이 성사 된다면 휴학을 해서라도 경력을 쌓는것이 좋음.
- 해커톤 전체 운영
  - github설정, Antigravity를 중심으로한 로컬 개발환경 셋팅을 1일차 오전중에 진행한다.
  - 희망하는 채용 공고를 한 개 찾는다
    - 현실성이 없어도 괜찮아. 가장 일하고 싶은 흥미로운 채용 공고를 찾아보자.
  - 비슷한 직군을 묶어서 조를 편성한다.
    - 조내에서 조장을 선출.
    - 조별로 슬랙 체널 생성
  - 90분을 하나의 스프린트로 해서 골을 세우고 진행한다.
    - 스프린트0는 1일차 오후 1시에 시작한다.
    - 75분간 각자 진행 후에 15분동안 팀내 리뷰를 진행한다.
    - 팀내 리뷰에서는 자신의 진행상황을 설명하고, 도움이 되는 아이디어를 공유한다.
