# AI 주도 포트폴리오 해커톤

### 3일 만에 실전 포트폴리오를 완성하는 여정

---

# 왜 AI인가?

- **2022년 11월 30일 ChatGPT 출시후 3년** 만에 세상이 완전히 변화
- 생성형 AI의 급격한 발전으로 산업 전반에 전례 없는 변화
- 기업들은 업무 방식을 재편하기 시작
- 채용 시장에도 큰 영향

---

# 주니어 포지션의 위기

- 많은 기업들이 **AI로 대체 가능한 업무** 정리 중
- 신규 채용 감소
- 특히 **신입·주니어 일자리의 감소**가 두드러짐

> 취업 준비생들에겐 경쟁 심화가 아닌
> **"게임의 규칙이 바뀐"** 상황

---

# 개발자와 데이터 분석가, 위험하다?

### AI 혁신의 파도

---

# 코딩의 종말

- 개발자의 **75%** 는 AI 코딩 도우미를 이미 사용 중
- 2년 안에 사람이 코드를 볼 필요성이 없어질 것으로 예상

> 구현은 AI가, 사람은 문제 정의와 해결에 집중

---

# 데이터 분석 직군도 예외 아님

- AI는 SQL 쿼리 작성, 대시보드 생성 같은 작업을 손쉽게 처리
- 오로지 기술적 작업에만 의존하는 분석가는 AI로 대체될 위험

**필요한 것:**
- 인사이트 도출
- 비즈니스 커뮤니케이션 능력
- 인간만의 강점

# AI는 위협이 아니라 든든한 동료

- AI를 두려워하는 대신 **내 편으로 만들기**

# 어떻게 AI를 내 편으로 만들까?

### Apply-Driven Development
### (지원 주도 개발 방법론)

---

# 문제에서 출발

- 전통적 bottom-up 학습: 기초부터 차근차근 (시간 오래 걸림)
- **Apply-Driven 방식**: 실제 문제를 출발점으로

> "필요한 것을 그때그때 배우며 만들어가는" 방식

---

# Top-Down 학습

1. 취업 목표(희망 직무)를 **먼저 정하기**
2. 그 직무에 필요한 기술을 **프로젝트를 만들며 습득**
3. 모르면 AI에게 즉시 물어보고 이해하면서 진행
4. **재귀적 갭 메우기 루프**

---

# 학습과 결과 동시 달성

```
먼저 대충이라도 만들어보기
       ↓
막히면 왜 막혔는지 AI에게 질문
       ↓
설명을 듣고 다시 시도
       ↓
반복!
```

**결과:** 작동하는 결과물 + 원리 완전 이해

---
